Detailed Description

OpenCL controller class.

Does framework initialization and is responsible for creation and management of basic objects that one should use to perform tasks on GPU device (or any other OpenCL compatible device).

Class is implemented as "singleton" so you can't instantiate it. Use shared() method to get a reference to the actual object.
